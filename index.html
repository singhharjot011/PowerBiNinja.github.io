Power BI Report On Canadian Immigration!

Main Objective of this Report

This Power BI report provies you User Freindly Interface to do in-depth analysis of Canadian Immigration from Year 2015-Present

More About My Report!

The Report is directly linked to IRCC website to obtain monthly data about new Permanent Residents based on their Age, Country of Origin, Current Residence (City and Province), NOC Category of Visa Applicant
Click here to access SourceData

What to Expect from this report

The Report provides different visuals, filters and Slicers that allows you to unveil the various statistics about top destination cities, NOC , Age Group along with numbers of Applicants
This report is great presentation of all the great features of Power BI from Slicers, Buttons, Drillthrough to Bookmarks and so on.

SQL Project on IPL Cricket League

Objective

This SQL Project combines various tables from dataset using SQL commands and functions to highlight High and Lows of Cricketers' Performance, teams statistics, and various other interesting statistics

Dataset Used

The Dataset has been downloaded from Data.World website. This data has statistics about each and every ball, match, result, players, umpire and every other detail about IPL matches from 2008-2016
Click here to access SourceData

Tools Used

This project is combination of SQL and PowerBI.
Microsoft PowerBI report is using direct query to extract data from Microsoft SQL Server Management Studio

Below is the screenshot from MS SQL SERVER MANAGEMENT that shows the top 3 rows of the tables used in project



Following is a list of the Questions that were answered using the database so far

  • Total Score of Winner Teams Sorted Highest To Lowest, along with venue, date and opposing team (Highest Scorer Teams)
  • Total Score of Looser Teams Sorted Lowest to Highest, along with venue, date and opposing team (Lowest Scorer Teams)
  • Total Number of Matches Played by each player over Years
  • Total Number of Matches Played by Each Team over Years
  • Total Number of Matches based on Stadium (Venue)
  • Fielder Caught highest to lowest number of catches (Best Fielder)
  • Top Six Hitter Batsman
  • Highest scorer Batsman to lowest scorer Batsman
